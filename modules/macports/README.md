Macports
========

Defines aliases and paths for macports on OSX.

Aliases
-------

### General

  - `pc` clean port tree
  - `pi` install a port
  - `psu` selfupdate ports
  - `puni` uninstall inactive ports
  - `puo` upgrade outdated ports
  - `pup` selfupdate and upgrade outdated ports
