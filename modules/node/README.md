Node.js
=======

Provides utility functions for [Node.js][1] and loads [npm][2] completion.

Also, provides [nvm][5]. nvm use default if you have default alias. 

Functions
---------

  - `node-doc` opens the Node.js online [API documentation][3] in the default
    browser.
